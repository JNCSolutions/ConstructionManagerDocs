Timesheet List

To access the timesheet list, select the Labour>Timesheet List tab on the main menu. This will open the following screen:



This displays a list of all timesheets entered on your system. They may have been entered manually, imported from a spreadsheet, or created from the Mobile Manager app.. The list can be manipulated in the usual way - see Manipulating Lists..



New

To create a new timesheet, click on the New tab at the top of the screen - circled in red above. This will open the following screen:



Data to be entered at header level includes:

  • Employee: Select from the drop-down list.
  • Date: Defaults to today, but should be changed to the week-ending date for the timesheet.
  • Set as Default Date: Tick this box if you want every line of the timesheet to default to this date. Leave un-ticked if you want to input multiple dates.
  • Details: Enter a description (optional).
  • Ref: Enter a reference (optional).

Now you can enter multiple lines of timesheet data - this includes:

  • M-job: Select from the drop-down list.
  • Job No.: Select from the drop-down list.
  • Tender Code: Select from the drop-down list, if applicable.
  • Date: Enter date, or accept the default.
  • Week: This is created automatically from the date. If week no. is incorrect, check the start date in Construction Manager (Desktop Version)>Settings>Defaults.
  • Details: Enter details of what work has been done - this will show in Job Costing.
  • Rate Description: Select from the drop-down list.
  • Rate Value: This will default to the rate held on the Pay Rates table, but can be over-written.
  • Hours: Enter hours worked.
  • Pay: Calculated from Rate X Hours.
  • Factor: Adjustment to allow for the cost of emplyment (e.g. NI and holiday time. A typical factor would be 1.30, but this is held on the Pay Rates table and can be over-written.
  • Cost: Calculated from Pay X Factor.
  • ChargeOut: Calculated from Hours X Chargeout Rate stored on the Pay Rates table. It can be over-written.

Rows can be added and deleted using the buttons circled in red, bottom left. Then Save your data, using the Save button, circled in red, top right.

N.B. See the section on Access Settings for information on controlling Payroll data access and visibility.


Edit

To edit an existing timesheet, select a line on the timesheet list and then open by either double-clicking on the line or select the Edit tab at the top of the screen - circled in red above. This will open a screen identical to the new timesheet screen.


Delete

To delete an existing timesheet, select a line on the timesheet list and then click on the Delete tab at the top of the screen - circled in red above. This will open a confirmation screen: click on either the Proceed or Cancel button.


Import

Timesheet data can be imported from an Excel spreadsheet - this is a more efficient method of data entry. To import timesheet data, click on the Import tab at the top of the main timesheet list screen. This will open the following screen:


Click on the import button - circled in red, bottom left. This will open the following screen, where you can select a timesheet to import, in the usual Microsoft manner:


Having selected a spreadsheet, click on the Import Spreadsheets button to proceed to the following screen:



Here you can either select the source columns from a drop-down list against each column, or use a template. This will tell the system which columns on the spreadsheet to feed into which columns on the timesheet (They may not be in the same order).

N.B. Take care to only tick the First Row is Headings box if that is the case, or you will miss the first row of data.

When you have selected columns, click on the Read Data button, the data will be read from the spreadsheet and the following screen will be displayed:


This screen displays the imported data: if you need to, you can edit the data at this stage (it is not saved yet). If you are happy with the data, click on the Save button and the data will be saved to the system.

N.B. in this example, only 1 record will be displayed on the main Timesheet List - select that line for edit, and the 4 detail lines will be displayed.
