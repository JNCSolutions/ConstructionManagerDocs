Timesheet List

To access the timesheet list, select the Labour>Timesheet List tab on the main menu. This will open the following screen:



This displays a list of all timesheets entered on your system. They may have been entered manually, imported from a spreadsheet, or created from the Mobile Manager app.. The list can be manipulated in the usual way - see Manipulating Lists..



Timesheet List

To create a new timesheet, click on the New tab at the top of the screen - circled in red. This will open the following screen:



Data to be entered at header level includes:

  • Employee: Select from the drop-down list.
  • Date: Defaults to today, but can be changed.
  • Set as Default Date: Tick this box if you want every line of the timesheet to default to this date. Leave un-ticked if you want to input multiple dates.
  • Details: Enter a description (optional).
  • Ref: Enter a reference (optional).

Now you can enter multiple lines of timesheet data - this includes:

  • M-job: Select from the drop-down list.
  • Job No.: Select from the drop-down list.
  • Tender Code: Select from the drop-down list, if applicable.
  • Enter date, or accept the default.
  • Week: This is created automatically from the date. If week no. is incorrect, check the start date in (old) Construction Manager>Settings>Defaults.
  • Details: Enter details of what has been done - this will show in Job Costing.
