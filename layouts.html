Introduction

ConstructionManager.Net comes supplied with a selection of standard reports for each module of the system. The Layouts module allows you to duplicate the standard reports and then modify them (the standard reports cannot be modified). These modified reports can then be used instead of, or as well as, the standard reports.

For the purpose of this help text, we will be using the Quotes module as an example.

To access the Layouts module, select the Layouts tab on the main menu, and then the Quotes tab - see below, circled in red. This will open the following screen:




New

To create a new timesheet, click on the New tab at the top of the screen - circled in red above. This will open the following screen:



Data to be entered at header level includes:

  • Employee: Select from the drop-down list.
  • Date: Defaults to today, but can be changed.
  • Set as Default Date: Tick this box if you want every line of the timesheet to default to this date. Leave un-ticked if you want to input multiple dates.
  • Details: Enter a description (optional).
  • Ref: Enter a reference (optional).
