Products

Products are used in your quotes. They can be created from ConstructionManager.Net, uploaded from a spreadsheet, or imported from Sage. Products are accessed via the Product tab on the main menu

The Products List




From the Product List you can:

  • Add Products
  • Edit Products
  • Delete Products
  • Import Products from Spreadsheet
  • Import Products from Sage

In the usual way, the user can search for products on the list using filters, manipulate columns to display and download the list to a spreadsheet or PDF file (see lists section).


Add Products

Create a new product by selecting the "New" tab at the top of the screen.




A blank screen will be displayed for the user to enter data into. All fields are optional apart from the product code.

All fields will used as default values when a quote is created.

The user can set a product as a Text Item or Term. When used on a quote, any cost information will be ignored and it will be printed without prices.

When the data has been entered, the user should click the "Save" button (circled in red). Then click the back arrow to return to the refreshed list.


Edit Products

Existing Products can be edited by selecting a product row and then clicking the "Edit" tab at the top of the screen.

A screen is then displayed for the user to modify data - This is identical to the "New" product screen, except that the Product No. cannot be changed. All fields are optional apart from the product code.

When the data has been entered, the user should click the "Save" button (circled in red). Then click the back arrow to return to the refreshed list.


Delete Products

Delete a product by selecting a Product row and then clicking the "Delete" tab at the top of the screen.




The tab will then turn red and the tab text will change to "Confirm Delete".

Click again to confirm, and the product will be deleted. The refreshed product list will then be re-displayed.

Click the "Import from Sage" button on the menu bar, and Construction Manager should begin importing your stored products from Sage. When it is finished, the page should refresh, and your products will be visible.

Importing Products from Spreadsheet

To import your products from a spreadsheet, click "Import from Spreadsheet". You will be taken to an entry screen where you can manually type in multiple products to import. However, click the "Import" button at the bottom, and you can select a spreadsheet to import your data from.

After selecting a spreadsheet and clicking "Import", a dialog will open you asking for the following details:

  • Which rows would you like to upload? By default, the start and stop values contain the entire spreadsheet. Unless you just want to import a subset of the sheet, you should leave these values alone.
  • Skip first row? If the first row of your spreadsheet contains a text header describing the columns, you should select this to avoid treating the column names as data.
  • Column assignments. Here you can assign columns from the imported sheet to the expected input fields.

After you've finished with these settings, simply confirm your selection and your data will be imported into the database.

Why are my products out of sync with Sage?

Because of the typically large quantities of data stored in the Sage products table, we do not pull this information through automatically. If you have made changes to your Sage data that you wish to be reflected on ConstructionManager.Net, you must perform the import process again. Your products will then be updated with the latest values from Sage.
