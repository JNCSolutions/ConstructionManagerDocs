Products are used in your quotes. They can be created from ConstructionManager.Net, uploaded from a spreadsheet, or imported from Sage. Products are accessed via the Product tab on the main menu

The Products List




From the Product List you can:

  • Add Products
  • Edit Products
  • Delete Products
  • Import Products from Spreadsheet
  • Import Products from Sage

In the usual way, the user can search for products on the list using filters, manipulate columns to display and download the list to a spreadsheet or PDF file (see lists section).


Add Products

Create a new product by selecting the New tab at the top of the screen.




A blank screen will be displayed for the user to enter data into. All fields are optional apart from the product code.

All fields will be used as default values when a quote is created.

The user can set a product as a Text Item or Term. When used on a quote, any cost information will be ignored and it will be printed without prices.

When the data has been entered, the user should click the Save button (circled in red). Then click the back arrow to return to the refreshed list.


Edit Products

Existing Products can be edited by selecting a product row and then clicking the Edit tab at the top of the screen.

A screen is then displayed for the user to modify data - This is identical to the "New" product screen, except that the Product No. cannot be changed. All fields are optional apart from the product code.

When the data has been entered, the user should click the Save button (circled in red). Then click the back arrow to return to the refreshed list.


Delete Products

Delete a product by selecting a Product row and then clicking the Delete tab at the top of the screen.




The tab will then turn red and the tab text will change to Confirm Delete.

Click again to confirm, and the product will be deleted. The refreshed product list will then be re-displayed.


Import Products from Spreadsheet

To Import Products from Spreadsheet click the tab at the top of the screen.

This will display a screen of blank product data rows.




The user can now:

  • Enter product data into the blank rows and click the Save button, OR
  • Click the Import button.

Clicking the Import button will display a screen where the user can select a spreadsheet to import from.




Choose a spreadsheet in the standard Microsoft fashion and click the Import Products" button.. Another screen will then be displayed.





After selecting a spreadsheet and clicking Import, a dialog box will open asking the user for the following details:

  • Which rows would you like to upload? By default, the start and stop values contain the entire spreadsheet. Unless the user just wants to import a subset of the sheet, leave these values alone.
  • Skip first row? If the first row of the spreadsheet contains a text header describing the columns, select this to avoid treating the column names as data.
  • Column assignments. Here the user can assign columns from the imported sheet to the expected input fields, OR click the "Template" button to assign columns via a template (see below).

Now click the "Upload" button. The following screen will be displayed:


This displays a list of products from the spreadsheet, ready to import into the main Product List. Click on the save button to finish the process. The new product list will now be re-displayed.

After clicking the Template button, the following screen will be displayed:




This allows the user to create, delete or use a selected template for column assignments.


Import Products from Sage

To Import Products from Sage click the tab at the top of the screen.

This will display a Please wait screen followed by a refreshed product list.
