Introduction

To access the planner, click the 'Planner' button on the row of options along the top, in blue. This will open the following screen:




Setup

There are 3 settings that need checking before Planner is implemented:

  • Employees - in Settings > Connection settings. Check this if you use Sage Payroll employees. If no employees are showing they will need setting up in Payroll or Construction Manager. Support can advise / assist with this.
  • Warning time periods (used with Maintenances Jobs and mobile manager) (see below in "Creating Appointments") – go to Login > Settings > Defaults > Maintenance
  • Working Period days and nights can be set in Settings > Defaults > Maintenance. This enables the night period and weekend to be hidden to allow more space on the planner screen. The non-working period can be displayed by clicking Options – Hide nightime / weekend hours. Use Sage Payroll Employees - in (new) ConstuctionManager.Net under Settings > Connection settings. Check this if you want to include employees from Sage payroll.

Screen Setup




Filter Options

Use the Filter options tab to control the view options on contracts, employees, sub-contractors, plant, custom resources and skills.


Contracts


Use the check boxes to control which contracts are to be viewed and hidden. You can also change the colour that is displayed with a contract. Normally all contracts or one contract are selected.


Employees, sub-contractors, plant and custom resources


Use the check boxes to control which records should be viewed and hidden. All 4 record types are controlled in the same way: i.e. Employees , Subcontractors , Plant , Custom Resources.


Skills


Use the check boxes to control which skills should be viewed and hidden. These skills are maintained on the employee database in Construction Manager.


Adding Custom Resources



If you need to plan additional resources - e.g. a gang of labour or multiple employees for one subcontractor - you can use Custom Resources. Each of these could be set up as a resource and planned individually. Custom Resources will appear on the left, beneath Employees, Sub-contractors and Plant.


Month, week, day view

You can elect to view the calendar either in Month, Week or Day view, using the appropriate tabs.



Screen View and Control




Time Scale Slider

You can use this feature to control the time scale - i.e. the size of display of days / weeks / months. This can be used in conjunction with the zoom featrure on your browser to aid visibility.



Go To Date

You can use this feature to position the screen to a particular date.

In addition, to move the grid, you can click and drag the timeline (date headings).


Options


Their are 3 sections to this feature:

  • The "General" tab
    • The "Row Scale" slider controls the vertical depth of the grid.
    • The "Compress Unworked Hours" option allows you to control whether the non-worked days and hours - typically, evenings and weekends - are displayed.
    • The Timeline Skip Days per Week" feature controls the number of days the week / day buttons skip (see below).
  • The "Job Display" tab. Here, you can control what fields from a job are displayed on an appointment.
  • The "M-Job Display" tab. Here, you can control what fields from a maintenance job are displayed on an appointment.

Day / Week arrows

You can use this feature to move the screen display forwards or backwards, by a week or day.


Employees, Subcontractors, Plant and Custom Resources



These components of the Planning system appear down the left-hand side of the grid. They can be hidden and displayed by clicking on any of the 4 headings. For instance: if you are planning sub-contractors, click on the employee heading to hide all employees. Click again and employees will re-display.


Creating and Editing Appointments

Create and Edit Appointments


To create an appointment, click in the row at the time the entry is to start, and then drag to where you want it to finish. This then opens the 'Create New Appointment' window, where you can choose whether the entry relates to a contract (1) or a maintenance job (2).

1. If a contract is picked, just click Save and the job will be planned in a purple box. If you want to add any notes enter these in the 'Instructions' box.

2. If a maintenance job is picked, the status must be entered as a visit is created as well as a planner entry. If you are using Mobile Manger the instructions should be filled in and these are the instructions that go to the mobile phone. Maintenance job appointments are created in a green block, and have a 'Warning system'. The colour can be set to change to amber X hours before the start time. The time period is set in Settings > Defaults > Maintenance tab > Planner appointment due warn (hours). When the start time has passed, the block turns red if the status has not been changed to "on site". To edit the details in the appointment, simply double-click the appointment block and the 'Edit Appointment' window will appear.


Moving and Re-assigning Appointments

To change the time slot of the planner entry, you can either click and move either side of the block (which resizes the block), or click the block in the middle and drag it to the correct time slot.

To change the person that the appointment is assigned to, click the block and drag it onto the correct row. You will then be asked if you want to change the visit reference number.


Copy and Delete Appointments

To copy an entry, right-click on the entry and click 'copy'. You can then move the cursor to a new position, and left-click will create a new appointment.

To delete an entry, right-click on the entry and click 'delete'.
