Introduction

To access the planner, click the 'Planner' button on the row of options along the top, in blue. This will open the following screen:



Setup


There are 2 settings that need checking before Planner is implemented:

  • Waiting periods (see below in "Creating appointments") - this is maintained in (old) Construction Manager under Settings > Defaults > Maintenance
  • Use Sage Payroll Employees - in (new) ConstuctionManager.Net under Settings > Connection settings. Check this if you want to include employees from Sage payroll.

Screen Setup



Adding Custom Resources



Custom resources can be added by the user. This would typically occur where a syb-contractor emplys a number of labourers - each of these could be set up as a resource and planned individually. Custom resources will appear on the left, beneath Plant.



Month, week, day view

CThe user can elect to view the calendar either in Month, Week or Day view, using the eppropriate tabs./p>

Filter options



Use the Filter options tab to control the view options on contracts, employees, sub-contractors, plant, custom resources and skills.


Contracts


Use the check boxes to control which contracts are to viewed and hidden. The user can also change the colour that is displayed with a contract - e.g. very important contracts could be changed to red.


Employees, sub-contractors, plant and custom resources


Use the check boxes to control which records should be viewed and hidden. All 4 record types are controlled in the same way.


Skills


Use the check boxes to control which skills should be viewed and hidden. These skills are maintained on the employee database in Construction Manager.

Screen View and Control



Time Scale Slider

The user can use this feature to control the time scale - i.e. the size of display of days / weeks / months. This can be used in conjunction with the zoom featrure on your browser to aid visibility.


Go To Date

The user can use this feature to position the screen to a particlar date.


Options


Their are 3 sections to this feature:

  • The "General" tab
    • The "Row Scale" slider controls the horizontal depth of the grid.
    • The "Compress Unworked Hours" option allows the user to control whether the non-worked days and hours - typically, evenings and weekends - are displayed.
    • The Timeline Skip Days per WQeek" feature controls the number of days the week / day buttons skip (see below).
  • The "Job Display" tab. Here, the user can control what fields from a job are displayed on an appointment.
  • The "M-Job Display" tab. Here, the user can control what fields from a maintenance job are displayed on an appointment.

Day / Week buttons

The user can use this feature to move the screen display forwards or backwards, by a week or day.


Employees, Subcontractors Plant and Custom Resources



These components of the Planning system appear down the left-hand side of the grid. They can be hidden and displayed by clicking on any of the 3 headings. For instance: if the user is planning sub-contractors, click on the employee heading to hide all employees. Click again and employees will re-display.



Original stuff

Dates are arranged chronologically along the horizontal axis at the top. The timescale can be changed either by clicking on the preset 'Month', 'Week' or 'Day' buttons (under 'Contracts'), or by toggling the 'Time to Scale' slider.

To move through dates, there are three different options:

  1. Click and drag the timeline (date headings).
  2. Jump days/weeks using the arrow buttons on the top left.
  3. Choose a specific date using 'Go To Date' button.

Filters

The filters above the timeline can also be used to quickly search through the rows to find the relevant entry. It is important to note that multiple selections can be input into the filter.

Creating, Editing & Deleting Appointments

Appointment Creation

To create an appointment in the planner, click in the row at time where you want the entry to start, and then drag to where you want it to finish. This then opens the 'Create New Appointment' window, where you can choose whether the entry relates to a contract (1) or a maintenance job (2). If you accidentally pick job from the wrong dropdown list, just click on that dropdown again and select the blank row at the top.

1. If you pick a contract, just fill in the 'Instructions' box with the information you want to appear in the entry. Then click Save and the appointment will be generated in a purple box.

2. If you pick a maintenance job, as well as filling out 'Instructions' for the appointment, a status can be set for it. Also, a 'Visit Ref' is automatically generated when the appointment is saved.

Maintenance job appointments are created in a green block, and having 'Waiting Periods'. These are periods over which the appointment is due (during which the block turns amber). When the due period has passed, the block turns red. The period can be set in the Construction Manager program on your computer under Settings > Defaults > Maintenance tab > Planner appointment due warn (hours).

Appointment Editing

To change the time slot of the planner entry, you can either click and move either side of the block (which resizes the block), or click the block in the middle and drag it to the correct time slot.

To change the person that the appointment is assigned to, click the block and drag it onto the correct row. You will then be asked if you want to change the visit reference number.

To edit the details in the appointment, simply double-click the appointment block and the 'Edit Appointment' window will appear.

Appointment Deletion

To delete an entry, right-click on the entry and click 'delete'.

Options

The 'Options' button opens a window with a list of options, explained below. There are three tabs: General, Job Display and M-Job Display.

Row Scale

The row scale slider determines the height of each row and, thereby, the height of the appointment blocks.

Compress Unworked hours

This tickbox allows you to choose whether or not nights and weekends (or whatever times are not counted 'Work Hours') are shown on the timeline. The day start and end can be set in Construction Manager in Settings > Defaults > Maintenance tab.

Timeline Skip Days Per Week

Chooses how many days to skip forward when clicking the 'Week' forward and backward arrow buttons.

Use Payroll Employeees

Option for whether or not to use the employees that have been set up in Sage Payroll
