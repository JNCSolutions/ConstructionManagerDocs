Introduction


Quotes are accessed via the Customer tab on the main menu




Before a quote is created, the user may wish to create a customer. This can be performed via 2 options under the Customer tab on the main menu.

The Sage Customers option creates customers in Sage, while the Quote Customers option creates customers purely in the Quotes sub-system. Both types of customer can be accessed for quotes.


The Quotes List


The quotes list displays all of your current quotes in a paginated, filterable list. From this list, you can:

  • Create a new quote
  • Edit existing quotes
  • Delete quotes
  • Print quotes
  • Email quotes
  • Export the list as a spreadsheet




The list can be filtered by entering selection data in the blank boxes above each column. Columns to view can be controlled by the "Columns" button (circled in red).



The Quote Screen

The quote screen has five tabs: Details, Items, Terms, Analysis and Notes.

Details

On the details tab, you control the information that you would normally associate with the quote header, such as the associated customer, contact details, address and date. You can also modify the header and description of works text.




The quote no. is auto-generated by the system.

Other data to be entered includes:

  • Customer: Either enter an existing quote customer or a Sage customer. Both can be selected from drop-down lists. Additionally, quote customers can be created at this stage, via the "+" button.
  • Branch: Select a branch from the drop down box of branches associated with the selected customer.
  • Header: Enter a short description of the job.
  • Decription of works: Enter a longer, more detailed description of the job. This is a multi-line, free form text field.
  • Footer: Enter any text that you wish to display at the foot of the quote.
  • Contact name: This defaults to the contact name set up on the customer master file, but can be overwritten.
  • Contact email: This defaults to the contact email address set up on the customer master file, but can be overwritten.
  • Contact tel: This defaults to the contact telephone no. set up on the customer master file, but can be overwritten.
  • Status: A new quote will default to Open, but can be changed later to Ordered, Lost or Inactive. An open quote that is converted to an order, via the button at the top of the screen, will be automatically changed to a status of Ordered. This field is used for reporting purposes.
  • Date: This defaults to today's date but can be overwritten.
  • Site address: This defaults to the customer's address but can be overwritten. Pick an address from the drop down list, or create a new address.


.

Items

The items tab contains the meat of the quote. Here you control the items that comprise the work itself.




Data to be entered on the items screen includes:

  • Stock code: There is a choice of 2 codes: M1, which will create a message text line - all costs will be ignored. Code Text will use costs to compile the quote. Pick a code from the drop-down list.
  • Description: Enter a short description.
  • Long Description: Enter a longer description.
  • Qty: Enter a quantity of the product being quoted.
  • Unit type: Enter a unit of measure (e.g. NO, Each, Kgs, Days etc.).
  • Cost price (£): Enter a cost price. This will be used either with the margin entry to calculate sales price, or with the sales price to calculate the margin.
  • Sales price (£): Enter a sales price to calculate the margin. OR:.
  • Margin (%): Enter a margin % to calculate the sales price.
  • Net (£): This is a calculated field for display only. It shows the calculated net value of the line
  • Tax code: Select a tax codd from the drop-down list. This will be used to calculate VAT on the line.
  • VAT (£): This is a calculated field for display only. It shows the value of VAT.
  • Total (£): This is a calculated field for display only. It shoes the total value of the line.


In addition, the following fields can be displayed using the columns button:

  • Tender code: A code entered here will feed through when the quote is allocated to a Job. It can then be used for advanced budget vs. costs reports.
  • Discount (%): Enter a discount%. This will reduce the calculated net value of the line.


The screen can be manipulated using the following Buttons (circled in red):

  • Move down: Highlight a line and press this button to move the line down.
  • Move up: Highlight a line and press this button to move the line up.
  • Delete: Highlight a line and press this button to delete the line.
  • Add item: Press this button to create a new line.


N.B. Once any items have been entered, the quote can be saved at any time by pressing the Save Button. This should be done frequently to minimise the chance of data loss.





Terms

The Terms screen gives you the opportunity to enter text that will appear at the end of the quote. You can access products that have been marked as text items, and pull through their associated text or simply key in text lines.. Terms can then be displayed separately from the items, at the end of your quote reports.




Data to be entered on the terms screen includes:

  • Code: Select a code from the drop-down box, or leave blank.
  • Short Description: This will default to the short description attached to the code, or you can over-write it. If no code is entered, you will have to key in a description.
  • Long Description: This will default to the long description attached to the code, or you can over-write it. If no code is entered, you will have to key in a description.


Analysis

The analysis screen allows you to enter reference and analysis data on a quote. This can be used for reporting and filtering on lists.




Data to be entered on the analysis screen includes:

  • Ref: A text reference for the quote.
  • Ex ref: An external text reference for the quote.
  • Classification: Select from a drop-down box of different types of job / work records.
  • Job analysis: Select from a drop-down box of different types of job analysis.
  • Project manager: Select from a drop-down box of different manager responsible for the job.

Notes

The notes screen allows you to enter time-stamped and signed notes onto a quote.




Data to be entered on the notes screen includes:

  • New Note: Enter note text and then press the Add Note button. Notes already entered will be displayed underneath.

Creating an Order from a Quote




When a quote is ready for progression, it can be converted to an Order using the Order button (circled in red). This will bring up the "Create Order" box, where you must attach the quote to a new or existing contract or a maintenance job / contract. If you wish to detach a quote from a contract, that must be done through the Contracts page.

Invoicing a Quote

Once a quote has been attached to a contract, it can be invoiced. This will create an invoice record in CM50, and optionally, in Sage. Once a quote has been invoiced, it can no longer be re-assigned to a different contract.
