The Quotes List

The quotes list displays all of your current quotes in a paginated, filterable list. From this list, you can go on to create a new quote, edit an existing one, print, email or export the list to an excel spreadsheet. Clicking new or choosing to edit an existing quote will take you to the quote screen.

The Quote Screen

The quote screen has four tabs: Details, Items, Analysis and Notes.

Details

On the details tab, you control the information that you would normally associate with the quote header, such as the associated customer, contact details, address and date. You can also modify the header and description of works text.

Items

The items tab contains the meat of the quote. Here you control the items that comprise the work itself. These items can either be inserted manually, or your sage products can be imported for use in constructionmanager.net.

Analysis

The analysis tab allows you to set the following values on a quote:

  • Ref: A text reference for the quote.
  • Ex ref: A text reference for the quote.
  • Classification: The type of job (e.g. residential, commercial).
  • Job analysis: Extra analysis tasks.
  • Project manager: The person responsible for managing the project.

Notes

The notes tab contains a list of timestamped and signed notes associated with a quote.

Invoicing a Quote

When a quote is ready for invoicing, it can be attached to a new or existing contract and invoiced directly. Once a quote has been invoiced, it can no longer be re-assigned to a different contract.
