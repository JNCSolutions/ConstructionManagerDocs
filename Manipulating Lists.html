Introduction


Lists are displayed in many places throughout Construction Manager, for selection and display of data. This section of Help Text explains how you can manipulate these list so that only the desired data is displayed. As an example, we will be using a list of Quotes, but all lists will share the same attrributes.


Selecting Values



Above every column on a list is one or, sometimes, 2 entry boxes (circled in red). You can key values, or partial values into these boxes, to select values from the column below.
