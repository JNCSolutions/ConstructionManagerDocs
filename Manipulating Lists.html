Introduction

Lists are displayed throughout ConstructionManager.Net for selection, display and reporting of data. This section explains the functions of these lists so that you can find, view and report on a specific selection of data. As an example, we will be using a list of Quotes, but all lists will share the same attributes.

On every list:

  • There will be a default setting for display order, for each user. This can be altered using the arrow at the top of any columns that display an arrow - see Sorting Data
  • There will be a default setting for which columns to display, for each user. This can be altered using the columns button at the bottom of the screen - see Selecting Columns
  • There will always be the following fields:
    • The user that created the data, and the date it was created on, and
    • The user that last modified the data, and the date it was modified on.
  • The list will automatically refresh when data is added or updated - i.e. if you create or update a quote on one browser, that data will automatically appear on another browser.
  • Column totals reflect the filtered list.

Scrolling through Data




If data fills more than 1 screen, then it can be searched by scrolling through in 2 ways:

  • Use the standard scroll bars on the side of the screen.
  • Use the Next, Last and Previous buttons or the numbered screen buttons to progress from one screen to another.

      • Selecting Values




      Above every column on a list is 1 or 2 entry boxes (circled in red, at top). You can key values, or partial values into these boxes, to select values from the column below.

      Some examples of this would include:

      • Above the status column: Enter - "Open" to select quotes at "Open" status.
      • Above the status column: Enter - "O" to select quotes at "Open" or "Ordered" status.
      • Above the Date Created column: Enter - "21/03/2016" and "22/03/2016" to select quotes created between those 2 dates.

      In addition, Filters can be used in conjunction with Selecting Values.

      You can use the filter button (circled in red, at bottom) to create or use a Filter. When data has been selected by entering a value at the top of a column, use the Filter button to save that Filter. Saved Filters can then be accessed and applied later. For instance, if you had selected Quotes at Status Open, then you could save that filter, access it later from the drop-down box, and Apply it.


      Sorting Data




      The first column on the quotes list has an arrow next to the heading (circled in red): This indicates which way the data is displayed - in ascending or descending sequence. Click on it to reverse the sequence. Any column that has an arrow next to the column name can be sorted in this fashion.


      Selecting Columns




      In the bottom left-hand corner of the screen is a button labelled "Columns" (circled in red): Click on this to display the Column Visibility selection box. You can then select or omit columns as desired. Additionally, you can alter the Rows per Page.


      Downloading Data to Excel and PDFs




      In the bottom left-hand corner of the screen are 2 buttons showing an Excel and Print icon (circled in red). These will download data to either an Excel file or a PDF print file - they work in an identical manner.

      Clicking on one of these buttons will display a box in the centre of the screen. You can then either select the Current Selection option which will download all the rows that you have highlighted or select the All Rows option to download all data in the list.

      The downloaded file name will then be displayed in the bottom corner of the screen in the normal Microsoft manner.
